## 六、货运调度 🚛

### 事件1：港口装货 🚢

**格式**：【货运调度🚛_港口装货🚢：装货【车牌号🚚】，司机【姓名👤】【电话📞】，关联【供应商👤】采购单价 XXX元💰每吨的【品牌📦】采购订单📋，装 XX吨⚖️， 关联供应商👤、采购订单📋和商品📦。】

---

### 事件2：到货处理 📦（卸货入仓 / 直送客户 / 销售送货 联动版）

**设计思路**：一车到货，根据业务需要可能有三种去向。当货物**直送客户**时（含"仅直送"和"入仓+直送"），系统在记录到货的同时**自动联动生成一条销售送货记录**，实现"车到即送"的闭环，避免二次录入。

---

表格

| 情形              | 格式                                                                                                                                                                                                                                                                                         |
| :-------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **仅卸货入仓** 📥    | 【货运调度🚛_卸货入仓📥：【车牌号🚚】，关联【供应商👤】采购单价 XXX元💰每吨的【品牌📦】采购订单📋，卸入仓库📥 XX吨⚖️，关联供应商👤、采购订单📋和商品📦。】                                                                                                                                                                                              |
| **仅直送客户** 🚚    | 【货运调度🚛_直送客户🚚：【车牌号🚚】，关联【供应商👤】采购单价 XXX元💰每吨的【品牌📦】采购订单📋，直发【客户👥】XX吨⚖️（对应每吨 XXX元💵的【品牌📦】销售订单📝），关联客户👥、销售送货🚛、销售订单📝、付款记录💸、采购订单📋和商品📦。】  <br>  <br><br>【销售送货🚛：送给【客户👥】约定单价为 XXX元💵每吨的【品牌📦】的销售订单 XX吨⚖️，送货方式是从车牌号XXX的货车【车上直发🚛】，关联客户👥、货物调度🚛、商品📦和对应订单📋。】                           |
| **入仓+直送** 📥+🚚 | 【货运调度🚛_卸货入仓📥+直送客户🚚：【车牌号🚚】，关联【供应商👤】采购单价 XXX元💰每吨的【品牌📦】采购订单📋，卸入仓库📥 XX吨⚖️，直发【客户👥】XX吨⚖️（对应每吨 XXX元💵的【品牌📦】销售订单📝），关联供应商👤、销售送货🚛、客户👥、采购订单📋、销售订单📝、付款记录💸和商品📦。】  <br>  <br><br>【销售送货🚛：送给【客户👥】约定单价为 XXX元💵每吨的【品牌📦】的销售订单 XX吨⚖️，送货方式是从车牌号XXX的货车【车上直发🚛】，关联客户👥、货物调度🚛、商品📦和对应订单📋。】 |


---

**联动规则说明**：

| 规则 | 说明 |
| :--- | :--- |
| **触发条件** | 只要记录中出现"直送客户"或"直发【客户】"，系统自动追加生成一条**销售送货**记录。 |
| **吨数同步** | 销售送货的吨数 = 直发客户的吨数（入仓+直送情形下，**不包含**入仓部分的吨数）。 |
| **单价同步** | 销售送货的单价 = 销售订单约定单价（💵），**不是**采购单价（💰）。 |
| **车牌同步** | 销售送货的车牌号 = 当前货运调度记录的车牌号，确保溯源一致。 |
| **品牌同步** | 销售送货的品牌 = 直发货物对应的品牌。 |
| **关联字段** | 销售送货自动关联：客户👥、本笔货物调度🚛、商品📦、对应销售订单📋。 |
| **不入仓不触发** | 仅卸货入仓📥的情形，无直发动作，**不生成**销售送货记录。 |
